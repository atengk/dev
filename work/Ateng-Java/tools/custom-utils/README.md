# 自定义工具类模块

## FileUtil

文件工具类



## FileTypeUtil

获取文件类型工具类

**添加依赖**

```xml
<!--文件类型工具类-->
<dependency>
    <groupId>org.apache.tika</groupId>
    <artifactId>tika-core</artifactId>
    <version>3.2.1</version>
</dependency>
```
